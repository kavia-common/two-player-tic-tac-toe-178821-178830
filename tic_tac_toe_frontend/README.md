# Tic Tac Toe - Ocean Professional

A lightweight React implementation of a two-player Tic Tac Toe game with a modern Ocean Professional theme.

## Features
- Two-player on the same device
- 3x3 grid with win/draw detection
- Turn indicator and accessible live status
- New Game (reset) button
- Responsive layout, subtle shadows, rounded corners
- No backend dependencies

## Quick Start
- npm install
- npm start
- Open http://localhost:3000

## Theme
Colors and styles live in `src/App.css`:
- Primary: #2563EB (blue)
- Secondary/Success: #F59E0B (amber)
- Error: #EF4444
- Background: #f9fafb
- Surface: #ffffff

## Notes
Environment variables are respected if present but not required. The app runs standalone.
